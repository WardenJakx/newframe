### Build
```bash
# Install
› bun install

# Build
› bun run build

# Typecheck (tsgo)
› bun run typecheck

# Rebuild on change
› bun run watch

# Layer dev server (Bun fullstack dev server with HMR)
› bun run serve
```

### Install

#### Chrome / Brave
1. Go to `brave://extensions` or `chrome://extensions`
2. Turn developer mode on if not already active (top-right corner)
3. Tap "Load unpacked"
4. Select the "dist" folder generated by running the building instructions above

#### Firefox
1. Go to `about:debugging#/runtime/this-firefox`
2. Tap "Load Temporary Add-on..."
3. Select `dist/manifest.json`
